1. Scaffolding is creating a folder with the initial files and folders to begin developing;a copy of the folder can be used as the starting template of a new project instead of starting with an empty folder.
2.Vite React is used to scaffold a new project with Vite. Vite is a tool that makes starting, developing, and building web apps much faster; provides boilerplate and scaffolding for common project types. Vite provides its own server for previewing websites.
